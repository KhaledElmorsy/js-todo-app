(inner, constant) containers

Consolidates generic containers for use by views.

Built as a psuedo-dictionary using getters to select elements when called by a new view's constructor. This circumvents containers that aren't in the DOM at page load (i.e. containers that are elements in other yet-to-be-rendered views) View
